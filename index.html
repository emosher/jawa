jawa - Just Another Web Application

This app serves to display the steps outlined in my presentation on HTML5 MVC Standards using AngularJS.

We will build an application to display information about FOSS projects. Angular will provide structure to our frontend JavaScript code as well as abstractions for common single page webapp functionality.

Steps in creating this app, with previews of each phase:

  • ng-app, data binding, and the app skeleton
  • ng-repeat with controller, app module
  • load data with $http
  • ng-view and routing with ngRoute
  • Library usage with Showdown.js and btf-markdown directive
